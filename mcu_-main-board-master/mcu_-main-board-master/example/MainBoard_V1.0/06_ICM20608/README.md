# ICM20608六轴传感器实验

本工程演示了如何使用IIC接口读取ICM20608六轴传感器数据。

## 硬件资源
- IIC接口
  - SCL - 时钟线
  - SDA - 数据线
- ICM20608传感器

## 功能说明
1. 传感器初始化配置
2. 读取加速度数据
3. 读取陀螺仪数据
4. 数据处理和滤波

## 使用说明
1. 编译并下载程序
2. 可以通过串口查看传感器数据
3. 支持数据实时输出
4. 可以进行姿态检测

## 注意事项
- 注意IIC地址配置
- 注意传感器的采样率设置
- 建议进行数据滤波处理
- 注意传感器的安装方向
- 可能需要进行零偏校准
