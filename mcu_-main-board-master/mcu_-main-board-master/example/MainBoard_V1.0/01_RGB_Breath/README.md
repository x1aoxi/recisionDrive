# RGB呼吸灯实验

本工程演示了如何使用PWM控制RGB LED实现呼吸灯效果。

## 硬件资源
- RGB LED (PWM控制)
  - RED   - TIM通道
  - GREEN - TIM通道
  - BLUE  - TIM通道

## 功能说明
1. 实现RGB LED的呼吸灯效果
2. 通过PWM调制实现LED亮度渐变
3. 可以实现单色或多色混合的呼吸效果

## 使用说明
1. 编译并下载程序
2. RGB LED将展现平滑的亮度渐变效果
3. 可以通过修改PWM参数调整呼吸效果

## 注意事项
- PWM频率设置需要合适，过低会导致闪烁感
- 注意RGB颜色混合的亮度控制
