# W25QXX Flash存储实验

本工程演示了如何使用SPI接口操作W25QXX系列Flash存储器。

## 硬件资源
- SPI接口
  - MOSI - 主机输出
  - MISO - 主机输入
  - SCK  - 时钟线
  - CS   - 片选线
- W25QXX Flash芯片

## 功能说明
1. Flash初始化和识别
2. 支持扇区擦除
3. 支持数据读写
4. 支持状态查询

## 使用说明
1. 编译并下载程序
2. 可以通过串口监视Flash操作
3. 支持数据的读写和擦除操作
4. 可以保存掉电数据

## 注意事项
- 注意擦除操作的时间
- 遵循写操作前擦除的规则
- 注意扇区大小和对齐
- 建议使用写保护功能
