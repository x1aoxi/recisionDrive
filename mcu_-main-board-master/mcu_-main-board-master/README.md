# 米醋电子工作室主控板

## 项目简介
本仓库包含米醋电子工作室开发的主控板系列产品的相关资料和例程。主控板采用STM32F103VET6芯片，提供完整的开发资源和使用说明。

## 目录结构
- `tools/` - 开发工具和固件
- `examples/` - 示例代码
- `docs/` - 技术文档
- `hardware/` - 硬件设计文件

## 开发环境搭建
1. 下载并安装以下工具：
   - WCHISPTool（烧录工具）
   - WCH-LinkUtility（调试工具）
   - STM32CubeMX（芯片配置工具）
   - Keil MDK（开发环境）
   
2. 按照 [tools/README.md](tools/README.md) 的说明配置烧录器

## 主控特性
- 芯片型号：STM32F103VET6

- 主频：72MHz

- Flash：512KB

- RAM：64KB

- 封装：LQFP100

1. 克隆本仓库
2. 按照开发环境搭建说明安装必要工具
