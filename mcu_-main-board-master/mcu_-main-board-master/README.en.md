# MCU MainBoard Project

## Introduction
This repository contains resources and example programs for the MCU development board series produced by MiCu Electronics Studio. The board uses the STM32F103VET6 chip and provides comprehensive development resources and documentation.

## Directory Structure
- `tools/` - Development tools and firmware
- `examples/` - Example code
- `docs/` - Technical documentation
- `hardware/` - Hardware design files

## Development Environment Setup
1. Download and install the following tools:
   - WCHISPTool (Programming tool)
   - WCH-LinkUtility (Debug tool)
   - STM32CubeMX (Chip configuration tool)
   - Keil MDK (Development environment)
   
2. Follow the instructions in [tools/README.md](tools/README.md) to configure the programmer

## MCU Features
- Model: STM32F103VET6
- Clock Speed: 72MHz
- Flash: 512KB
- RAM: 64KB
- Package: LQFP100

## Quick Start
1. Clone this repository
2. Set up the development environment following the instructions above
3. Use STM32CubeMX to generate project base code
4. Refer to example programs to start development
